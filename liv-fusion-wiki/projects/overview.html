1.3. 研究主线（Engineering + Research）

Phase A · 前端融合跑通

FAST-LIVO1/2：建立稳定实时估计主线，保证高频可用。

Phase B · 部署成本降低

FAST-Calib + LIV-Handheld：降低标定与采集门槛，提升复现效率。

Phase C · 全局一致性增强

Global-LVBA + LiDAR-Visual-RTK：补强长程收敛，服务大尺度重访。

Phase D · 表示与任务扩展

LIV-GaussMap + UMI：让地图不仅用于定位，也服务示教与操作学习。

1.4. 我关心的核心问题

  1. 如何在真实场景中同时做到 精度、鲁棒性、实时性？

  2. 如何让系统具备 可部署、可维护、可复制 的工程品质？

  3. 如何在低成本传感器组合下逼近高端系统效果（low-cost lifelong SLAM）？

  4. 如何把几何融合系统与 UMI / VLA / Diffusion Policy 打通？
