1. FAST-LIVO1

LIVO Baseline · Real-time · Deployable

FAST-LIVO1：实时激光视觉惯性紧耦合里程计

从“实验室可运行”走向“真实场景可部署”的关键里程碑。

  • 类型：LiDAR-Inertial-Visual Odometry

  • 阶段：已完成 / 代表作

  • 代码：https://github.com/hku-mars/FAST-LIVO

  • 论文：https://arxiv.org/abs/2203.00893

1.1. 一句话价值

FAST-LIVO1 建立了稳定可复现的 LIVO 工程基线，为后续 FAST-LIVO2、全局优化和具身扩展提供了可持续演进起点。

1.2. 核心亮点

  • LiDAR + Camera + IMU 的高频紧耦合融合链路

  • 在实时性与精度之间保持工程可用平衡

  • 面向部署的参数组织和运行流程

  • 可直接对接后续全局优化与表示模块

1.3. 支持平台（卡片化展示建议）

  • ✅ 通用 x86

  • ✅ Radxa ROCK5B+

  • ✅ OrangePi 5 Ultra

  • ✅ NVIDIA Jetson Orin Nano Super

  • ✅ NVIDIA Jetson AGX Orin

  • ✅ D-Robotics RDK X5

1.4. 传感器支持

  • ✅ LiDAR + Camera + IMU（硬同步优先）

  • ✅ AC1（示例）

  • ✅ 可扩展到同类时空同步传感器组合

1.5. 复现入口（最小可用）

1.5.1. 环境依赖

  • Ubuntu 16.04 ~ 20.04

  • ROS（推荐 Noetic）

  • PCL >= 1.6 / Eigen >= 3.3.4 / OpenCV >= 3.2

  • Sophus（README 指定版本）、Vikit、livox_ros_driver

1.5.2. 编译

cd ~/catkin_ws/src
git clone https://github.com/hku-mars/FAST-LIVO
cd ../
catkin_make
source ~/catkin_ws/devel/setup.bash

1.5.3. Benchmark 运行

roslaunch fast_livo mapping_avia_ntu.launch
rosbag play YOUR_DOWNLOADED.bag

1.5.4. 私有数据运行

roslaunch fast_livo mapping_avia.launch
rosbag play YOUR_DOWNLOADED.bag

1.5.5. 常见问题

  1. 轨迹漂移明显：先查时间同步、外参、IMU噪声参数。

  2. 实时性不足：调大 point_filter_num、调整滤波参数。

  3. 视觉模块不稳：按光照调整 outlier_threshold，检查时间对齐。
